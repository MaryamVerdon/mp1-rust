Présentation & Historique

Rust est un langage de programmation libre et compilé conçu par Mozilla Research et disponible sur Windows, MacOS et Linux. Il est multi-paradigme, plus précisément impératif (en séquence), fonctionnel, procédural et orienté objet sous certains aspects. Il est fiable et pratique. De plus, il a pour but d’être concurrent (prend en compte des threads et processus).

À la base développée par Graydon Hoare dans le cadre d’un projet personnel en 2006, son employeur, Mozilla, commence sa participation en 2009 et est actuellement son concepteur et développeur depuis 2010. Il passera d’un compilateur, initialement écrit en OCaml, à un compilateur écrit en Rust appelé rustc en 2011, avec une première version alpha en 2012. Rust aura sa en version stable (1.0) en 2015.

Il est aujourd’hui utilisé dans divers programmes système, applications en ligne de commande, applications web, services réseaux et systèmes embarqués.

Performance

Rust est terriblement rapide et économique au niveau de la mémoire. Il peut dynamiser des services à hautes performances et s'intègre facilement à d'autres langages.

Fiabilité

Le puissant système permet d'éliminer de nombreuses variétés de bugs dès la compilation.

Productivité

Rust dispose d'une excellente documentation, d'un compilateur bienveillant, avec des messages d'erreur utiles.

Les outils

Outils

Rust embarque tous les outils essentiels dont a besoin un développeur, centralisé à un seul et même endroit. Il dispose de :

  • rustup, un gestionnaire de toolchains avec rustc, cargo, rust-fmt, etc. (paquets pour la compilation et autres) et un gestionnaire de version pour chaque canal de release (nightly, beta, stable) via le GitHub de Rust afin d’installer et de garder à jour Rust.
  • cargo, un gestionnaire de paquets et surout un gestionnaire de projets qui permet un agencement modulaire des différents composants.

    • Ces outils permettent la génération rapide de projet, le développeur peut ainsi se mettre au travail rapidement avec le strict minimum.

    Plugin

    Rust dispose de son propre plugin à travers plusieurs IDE :

  • Visual Studio : VisualRust, et peut donc être utilisé comme n’importe quel projet C#, F#, C++, ou autre langage supporté par la plateforme.
  • Les IDE (Environnement de développement intégré) de JetBrains (PhpStorm...) : IntelliJ Rust.

    • Installation & Utilisation

    Pour utiliser Rust, Il faut donc au préalable installer rustup.

    Par la suite, nous pouvons créer un projet Rust via la commande cargo new nomProjet (Commencer Rust), puis cargo run (en étant dans le projet) pour compiler et lancer le programme. On peut éditer le code dans « src » dans le fichier « main.rs ».

    Nous pouvons aussi créer uniquement un fichier « .rs » pour un petit programme simple, que l’on compile avec la commande rustc nom.rs pour compiler et générer un fichier .exe que l’on peut exécuter dans le terminal nom.exe.
